🌍 Smart Tourism System

A full-stack tourism booking platform built using Django.
This system allows users to explore services, make bookings, and enables providers to manage their services and bookings efficiently.

---

🚀 Features

👤 User Features

- User Registration & Login
- Explore Categories & Subcategories
- View Services with Image Gallery
- Book Services
- View My Bookings
- Cancel Booking
- Add Reviews & Ratings
- Search Services
- Sort Services (Price / Newest)

---

🏢 Provider Features

- Provider Registration
- Profile Completion
- Add & Manage Services
- Upload Multiple Service Images
- View Bookings
- Accept / Reject Bookings

---

🛠 Admin Features

- Manage Categories & Subcategories
- Approve / Reject Providers
- Monitor System Activity

---

🛠 Tech Stack

- Backend: Django (Python)
- Frontend: HTML, CSS, JavaScript, Bootstrap
- Database: SQLite
- Tools: Git, GitHub

---

⚙️ Installation Guide

1️⃣ Clone Repository

git clone https://github.com/Sudeepssj/smart-tourism-system.git
cd smart-tourism-system

---

2️⃣ Create Virtual Environment

python -m venv venv
venv\Scripts\activate

---

3️⃣ Install Dependencies

pip install -r requirements.txt

---

4️⃣ Run Server

python manage.py runserver

Open in browser:
http://127.0.0.1:8000/

📊 Booking Flow

User → Select Service → Book Now → Confirm Booking
↓
Provider → Accept / Reject
↓
User → View Status / Cancel Booking
